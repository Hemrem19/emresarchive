# Logo Files

This directory should contain the citavErs logo files.

## Required Logo Files

### 1. Logo Mark (Icon Only)
- **File**: `logo-mark.png` or `logo-mark.svg`
- **Description**: The stylized "C" with arrow and book icon (no text)
- **Use**: Favicon, app icons, small size branding

### 2. Full Logo (Mark + Text)
- **File**: `logo-full.png` or `logo-full.svg`
- **Description**: The full "citaVErs" logo with graphic mark and text
- **Use**: Header, branding, documentation

### 3. Favicon
- **File**: `favicon.ico` or `favicon.png`
- **Description**: 32x32 or 16x16 icon version of the logo mark
- **Use**: Browser tab icon

### 4. App Icons (PWA)
- **File**: `icon-192.png` (192x192 pixels)
- **File**: `icon-512.png` (512x512 pixels)
- **Description**: Square app icons for PWA
- **Use**: Mobile home screen, PWA installation

## Logo Specifications

Based on the provided logos:
- **Primary Color**: Dark teal/blue-green (#137fec or similar)
- **Text**: "citaVErs" (mixed case: cita + V + Ers)
- **Graphic Elements**: 
  - Stylized "C" shape
  - Upward arrow
  - Book/bookmark icon

## Placement Instructions

1. Place the logo files in this `assets/logos/` directory
2. Ensure files are optimized for web (compressed PNG or SVG format)
3. Update `index.html` favicon and logo references to point to these files
